ระบบบันทึกหลังการจัดการเรียนรู้ เวอร์ชัน 3

สิ่งที่เพิ่ม/ปรับในเวอร์ชันนี้
- ใช้รูปแบบเอกสารเหมือนไฟล์ Word ต้นฉบับมากขึ้น
- ไม่มีโลโก้โรงเรียน
- ไม่มีช่องลายเซ็น
- เพิ่มช่อง "บรรทัดชั้น/กลุ่มเรียน" เพื่อให้พิมพ์ตามฟอร์มจริงได้ เช่น
  ช่างไฟฟ้ากำลัง ไฟฟ้ากำลัง/1 2565 (ชฟ.3/1 )
- เลือกดาวน์โหลด Word หรือ PDF ได้
- เพิ่มรายวิชาและห้องเรียนได้
- เก็บประวัติย้อนหลังตามรายวิชา/รายห้อง

วิธีติดตั้งครั้งแรก
1. แตกไฟล์ ZIP
2. ดับเบิลคลิก install_windows.bat
3. รอให้ติดตั้งเสร็จ

วิธีเปิดใช้งาน
1. ดับเบิลคลิก run_windows.bat
2. ถ้า Streamlit ถาม Email ให้กด Enter ผ่านได้เลย
3. เปิดลิงก์ http://localhost:8501

การใช้งาน
1. ไปที่เมนู จัดการรายวิชา/ห้องเรียน เพื่อเพิ่มรายวิชาและห้องเรียน
2. ไปที่เมนู กรอกบันทึกหลังสอน
3. เลือกวันที่ สัปดาห์ ครั้งที่สอน รายวิชา ห้องเรียน
4. กรอกเนื้อหา ปัญหา แนวทางแก้ไข และผลการจัดการเรียนรู้
5. กดบันทึกและสร้างไฟล์
6. ดาวน์โหลด Word หรือ PDF ตามที่เลือก

หมายเหตุ
- ไฟล์ที่ export จะอยู่ในโฟลเดอร์ exports
- ฐานข้อมูลอยู่ในโฟลเดอร์ data
- ถ้าต้องการสำรองข้อมูล ให้เก็บโฟลเดอร์ data ไว้
